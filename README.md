## Package Installation

### `pnpm install`

## App Start

### `pnpm run dev`

## In this project, you can see:

- ☑️ React

- ☑️ React Router
- ☑️ localStorage
- ☑️ React Hooks
- ☑️ Functional Components
- ☑️ Token based Authentication
- ☑️ Protected Routes
- ☑️ Ant Design
- ☑️ Ant Design Charts for data visualization
- ☑️ Pagination
- ☑️ Axios
- ☑️ CRUD operations
